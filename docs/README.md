## 🚀 구현할 기능 목록

---
자동차 경주 게임 구현
- 사용자 입력 안내 문구 출력
- 사용자 입력을 받기
  - 자동차 이름을 받아서 파싱한후 car 객체에 저장
  - 시도할 횟수를 받아서 저장
  - 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다.
- 자동차가 전진할지 말지 결정
  - 무작위 값을 구해 조건에 맞을 시 전진
- 각 차수별 실행결과 보여주기
- 경주를 마친 후 우승결과 출력
  - 우승자가 여러명일 경우 쉼표(,)를 통해 구분
